# Catan Dice

A beautiful, feature-rich dice roller built for Settlers of Catan. Runs entirely in the browser as a single HTML file.

**[Live Demo](https://gowtham3105.github.io/catan-dice/)**

## Features

- **Two-dice roller** with smooth animations and particle effects
- **Robber alert** — full-screen overlay when you roll a 7
- **Special roll detection** — Snake Eyes (2), Boxcars (12), and Doubles
- **Roll statistics** — tracks totals, averages, distribution chart, and roll history
- **Multiplayer turn tracking** — add 2-6 players with color-coded turn bar
- **Sound effects** — toggle on/off with dice shake and result sounds
- **Fully responsive** — works on desktop and mobile
- **No dependencies** — single `index.html` file, no build step required

## Usage

Open `index.html` in any modern browser, or visit the [live demo](https://gowtham3105.github.io/catan-dice/).

1. Tap **Roll Dice** to roll
2. Use the player panel to add players and track turns
3. View stats and roll history below the dice
